## Setup Instructions

## 1. Clone the repository

## 2. Go to client-app
npm install
npm run dev

## 3. Go to product-service
npm install
nest start run

## 4. Go to order-service
npm install
nest start run

## 5. Additional notes

- Go to swagger docs to hit the apis.
- Make sure both apps are running simultaneously during development.
- Check package.json scripts for any custom commands.
